Web-based chess with unique game modes and serverless P2P multiplayer.

**Live Demo:** https://ricardodeazambuja.github.io/chaos_chess/

## Game Modes

- **Normie** - Standard chess
- **Rotating Chaos** - Switch colors after each move
- **Random Chaos** - Random color assignment per turn
- **Network Play** - P2P multiplayer via WebRTC (no server needed)

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.
